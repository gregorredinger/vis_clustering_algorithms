Project Specification

In our Project we have the Goal to Visualize the Performance of different Clustering Algorithms. The Algorithms we will analyze are k-means, DBSCAN and Optics. We expect a significant Programming effort for our Project, so we decided to realize it as a Technique Project. The Application we create let the user choose a Clustering Algorithm and analyze different predefined Datasets. The User then can click through the Calculation Path of the Algorithm Step by Step and get a visual Feedback in Form of a Diagram that gets updated after each Step. We decided to use small Datasets because it's easier to Follow for a User what the Algorithm did.
